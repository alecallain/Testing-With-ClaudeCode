# Office Chores

> **Note:** This project was built as a test to evaluate Claude's skillset and capabilities.

A React app for managing recurring chores in an office setting. Team members can be added, chores created with recurrence rules, and completions tracked on a monthly calendar view.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 7** — dev server and bundler
- **Tailwind CSS 3** — utility-first styling
- **Headless UI** — accessible modal dialogs
- **date-fns 4** — date math and formatting

## Getting Started

```bash
npm install
npm run dev       # Start Vite dev server (HMR)
npm run build     # Type-check then build for production
npm run lint      # Run ESLint
npm run preview   # Preview production build locally
```
